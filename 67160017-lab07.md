1.6 ตอบคําถามต่อไปนี้เพื่อวัดความเข้าใจ:

1. querySelector vs querySelectorAll: อะไรคือความแตกต่าง
   o querySelector() ใช้เมื่อไร -> ใช้เมื่อต้องการเลือก class หรือ id อย่างเฉพาะเจาะจงมาตัวนึง
   o querySelectorAll() ใช้เมื่อไร -> ใช้เมื่อต้องการเลือก class มาทั้งหมดในเขตนั้น
2. classList.toggle(): nav.classList.toggle('hidden') ทํางานอย่างไร
   o ถ้า nav มี class 'hidden' แล้ว toggle() จะทําอะไร -> ลบ class "hidden" ออก
   o ถ้า nav ไม่มี class 'hidden' แล้ว toggle() จะทําอะไร -> เพิ่ม class "hidden" เข้าไป
3. event.target: เมื่อคลิกปุ่ม event.target จะชี้ไปที่อะไร -> จะชี้ไปที่ element ที่อยู่ในที่สุเ ขณะกำลังทำงาน
4. remove() vs toggle(): เมื่อคลิก link ควร
   ใช้ nav.classList.remove('hidden') หรือ nav.classList.toggle('hidden')? ทําไม -> ใช้ toggle จะดีกว่าเพราะว่าถ้าแค่ remove จะไม่สามารถเพิ่มใหม่ได้ด้วยตัวมันเอง
5. display: none: .hidden { display: none; } ทําไมจึงจําเป็น? วิธีอื่นเพื่อซ่อน element ได้หรือไม่ -> hidden มันแค่ชื่อของ class สามารถใช้คำอื่นได้ แต่ display:none ใช้สำหรับไม่แสดงผล ซึ่งสำคัญ
